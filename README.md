## 👨🏻‍💻 Development Steps

1. **Locally test and develop**

Once the repo is cloned on your local environment, you can instantly begin developing!

2.  **Start developing.**

    Navigate into your new site’s directory and start it up.

    ```shell
    cd soaapp.github.io/
    npm run develop
    ```

3.  **Open the code and start customizing!**

    Your site is now running at http://localhost:8000!

    Edit anyt `.tsx` file to see your site update in real-time.

## 🚀 Deployment Steps

1.  **Deploy soaapp's portfolio to GH Pages.**

    Use the Gatsby CLI to create a new site, specifying the minimal TypeScript starter.

    ```shell
    # deploy with script in package.json
    npm run deploy
    ```
